# SistemaMensajes
Sistema notificador de mensajes con arduino e interfaz de java


El sistema consta de:

* Sensor de temperatura
* Sensor de humedad
* Sensor de luminosidad
* Display LCD 16x2
* Potenciometro 10k
* Arduino Uno
* Cables
